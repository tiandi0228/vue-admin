# Vue-Admin

基于Vue3.4、TypeScript、Vite5、Pinia、antv/g2、 Element Plus开源的一款后台管理框架

### 使用方法

#### clone

```
git clone https://github.com/tiandi0228/vue-admin.git
```

#### install

```
pnpm install
```

#### run

```
pnpm dev
```

#### build

```
pnpm build
```
